## نظرة عامة

تطوير ثيم قمرة يتطلب:

1. **تثبيت CLI** - أداة سطر الأوامر
2. **تسجيل الدخول** - ربط حسابك
3. **إنشاء الثيم** - البنية الأساسية
4. **التطوير** - معاينة محلية
5. **النشر** - رفع الثيم

## التقنيات المطلوبة

- Node.js (v16 أو أحدث)
- npm أو yarn
- محرر نصوص (VS Code موصى به)
